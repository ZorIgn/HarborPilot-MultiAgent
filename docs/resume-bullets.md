# Resume Bullets

- Built HarborPilot AI, a full-stack multi-agent admissions planning platform for Hong Kong/Singapore master's applications using FastAPI, Next.js, typed Agent contracts, deterministic rule gates, and a pluggable LLM adapter.
- Implemented an orchestrated workflow across Profile, Evidence, Evaluation, Program Research, Matching, Timeline, Writing, and Review agents, returning auditable traces, human gates, and source-aware recommendations.
- Designed a rule-first matching engine that prevents hard-condition violations, separates self-reported from confirmed evidence, and avoids fabricated admission probabilities.
- Created a portfolio-ready product UI with background assessment, project recommendations, case intelligence, material verification, writing review, and data governance modules.
- Added CI, Docker Compose, synthetic 2027-Fall program data, rule DSL examples, architecture docs, and automated tests for workflow order and eligibility behavior.
